‎🖥️ Usage
‎
‎After starting the tool, you will see:
‎
‎1. Enter Target URL – Your own server URL (e.g., https://yourserver.com)
‎2. Choose Method – GET or POST
‎3. Set Threads – 1 to 500 (higher = heavier)
‎4. Set Duration – Seconds to run the test
‎5. Confirm – Type y to start
‎
‎Example
‎
‎```
‎Target URL: https://my-test-server.com
‎Method: GET
‎Threads: 50
‎Duration: 30s
‎```
