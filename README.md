## 📌 Description

**DDOS ENGINE** is a professional stress-testing tool built for **educational purposes** and **testing your own server**. It allows you to simulate HTTP flood attacks to measure your server's resilience.

> ⚠️ **WARNING:** This tool is intended for **testing your OWN server only**.  
> The developer is **NOT responsible** for any misuse.  
> Using this tool on servers you do not own is **ILLEGAL**.
